# Frappe Inspector Support

Public support, documentation and issue tracking for Frappe Inspector.

Use this repository for:

- Bug reports.
- Feature requests.
- Compatibility problems.
- False-positive diagnostics.
- Performance problems.
- Public documentation.
- Public changelog notes.
- License and privacy information.

## Issue Types

- Bug report.
- Feature request.
- Compatibility problem.
- False-positive diagnostic.
- Performance problem.

Do not post secrets, private source code, customer data, database dumps, API keys, or paid-license information in public issues.
