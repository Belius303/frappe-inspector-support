# Privacy

Frappe Inspector analyzes project files locally inside the IDE process.

- Telemetry is disabled by default.
- Source code, customer names, full paths, database content and business data are not uploaded.
- The plugin does not run `bench`, import project Python modules or execute project code during indexing.
- License checks use JetBrains Marketplace licensing mechanisms.
- Bug reports are created by users and should be reviewed before sharing.
